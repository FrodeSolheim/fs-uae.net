Download

Windows

Download the two .zip archives and extract the contained FS-UAE and FS-UAE-Launcher folders to wherever you like. You must keep these two folders stored alongside each other. The fs-uae-launcher.exe executable is found within FS-UAE-Launcher/Windows/x86-64. A Windows installer is currently not available.

x86-64
FS-UAE_3.2.35_Windows_x86-64.zip
FS-UAE-Launcher_3.2.20_Windows_x86-64.zip

You may also want to install plugins for IPF or PPC support.

macOS

Download the .dmg images for both FS-UAE Launcher and FS-UAE, and copy the apps within to /Applications or wherever you like. You must keep these two apps stored alongside each other.

You also need to choose the correct version for your machine; ARM64 for Apple silicon, and x86-64 for Intel. The ARM64 version will run faster on Apple silicon, but does not support JIT compilation. You should be able to run the x86-64 on M1 Macs (with JIT) though of course performance will be affected somewhat by emulating both 68K and x86-64 on ARM64 hardware.

x86-64
FS-UAE_3.2.35_macOS_x86-64.dmg
FS-UAE-Launcher_3.2.20_macOS_x86-64.dmg

ARM64
FS-UAE_3.2.35_macOS_ARM64.dmg
FS-UAE-Launcher_3.2.20_macOS_ARM64.dmg

You may also want to install plugins for IPF or PPC support.

Linux

Download the .tar.xz archives for your operating system and extract the contained FS-UAE and FS-UAE-Launcher directories to wherever you like. You must keep these directories stored alongside each other. The fs-uae-launcher executable is found within FS-UAE-Launcher/Linux/x86-64.

x86-64
FS-UAE_3.2.35_Linux_x86-64.tar.xz
FS-UAE-Launcher_3.2.20_Linux_x86-64.tar.xz

ARM64
FS-UAE_3.2.35_Linux_ARM64.tar.xz
FS-UAE-Launcher_3.2.20_Linux_ARM64.tar.xz

You may also want to install plugins for IPF or PPC support.

Alternatively, if you are running Debian, Ubuntu, Fedora or openSUSE, you may instead use my official package repositories and install packages built specifically for your Linux distribution (see further below).

Ubuntu & Debian

To get the latest version, first visit my Ubuntu or Debian package repository and add it to your system using the commands listed there. If you skip this step you may get an outdated version from your Ubuntu or Debian distribution. Then, to install FS-UAE as well as the Launcher and the Arcade frontends: 

sudo apt update
sudo apt install fs-uae fs-uae-launcher fs-uae-arcade

You may also want to install plugins for IPF or PPC support.

Fedora

To get the latest version, first visit my Fedora package repository and add it to your system using the commands listed there. If you skip this step you may get an outdated version from your Fedora distribution. Then, to install FS-UAE as well as the Launcher and the Arcade frontends: 

sudo dnf install fs-uae fs-uae-launcher fs-uae-arcade kernel-modules-extra

You may also want to install plugins for IPF or PPC support.

openSUSE

To get the latest version, first visit my openSUSE package repository and add it to your system using the commands listed there. If you skip this step you may get an outdated version from your Fedora distribution. Then, to install FS-UAE as well as the Launcher and the Arcade frontends: 

sudo zypper install fs-uae fs-uae-launcher fs-uae-arcade

You may also want to install plugins for IPF or PPC support.
