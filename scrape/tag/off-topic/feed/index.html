GCC 4.x does no longer support compilation for AmigaOS/m68k, so being able to easily install GCC 2.95 again is very nice. Compiling the old GNU tools on modern systems is not easy, so I’m sure Krystian has put a lot of effort into this -so thanks The cross-compiler environment contains all required Amiga headers and libraries, and vbcc too!

To make it even easier to use, I have published pre-compiled Linux binaries. I plan to follow up with OS X binaries later.
