Debian

To get the latest version, first visit my Debian package repository and add it to your system using the commands listed there. If you skip this step you may get an outdated version from your Ubuntu or Debian distribution. Then, to install FS-UAE as well as the Launcher and the Arcade frontends: 

sudo apt update
sudo apt install fs-uae fs-uae-launcher fs-uae-arcade

Alternatively, you can download the generic Linux version which should work on most recent Linux distributions.
