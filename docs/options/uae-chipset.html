Documentation

Option: uae_chipset

Category: Hardware
Default:
Example: aga
Overrides the chipset type in the emulated Amiga. You normally don't need
to use this option since a suitable chipset is automatically chosen based
on Amiga model. Valid values are:
Values:
* ocs (OCS)
* ecs (ECS)
* ecs_agnus (ECS Agnus)
* ecs_denise (ECS Denise)
* aga (AGA)
Code:
Code:
