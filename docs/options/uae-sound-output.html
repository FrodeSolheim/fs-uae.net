Documentation

Option: uae_sound_output

Category: Hardware
Default:
Description: Sound emulation
Example: interrupts
Allows you to choose whether sound is emulated or not. Valid values are:
Value: none (Disabled)
Value: interrupts (Emulated, No Output)
Value: exact (Enabled)
