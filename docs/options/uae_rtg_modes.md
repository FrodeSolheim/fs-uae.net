description: RTG color modes
default: 0x212
type: Choice
---

The value is really a bit mask, and other combinations are available.

Value: 0x212 (8bit + 16bit PC + 32bit BGRA)
Value: 0x602 (8bit + 16bit + 32bit BGRA)
Value: 0x112 (8bit + 16bit + 32bit RGBA)
