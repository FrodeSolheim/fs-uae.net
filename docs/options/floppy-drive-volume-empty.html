Option: floppy_drive_volume_empty

Description: "Empty Floppy Drive Volume"
Type: integer
Default: 25
Example: 0
Range: 0 - 100
Set volume to 0 to disable floppy drive clicks when the drive is empty.
Max volume is 100.
