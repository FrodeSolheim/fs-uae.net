Documentation

Option: load_state

Description: Load state by number
Default: 
Example: 5
Type: integer
Range: 1 - 9
Since: 2.3.12
Loads a save state identified by the save state slot number (1..9) when FS-UAE
starts. If the state does not exist, nothing will happen.
