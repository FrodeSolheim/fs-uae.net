Option: cpu

Summary: "CPU"
Type: Choice
Default:
// Default: auto
Example: 68040
Since: 2.5.19
Use this option to override the CPU model in the emulated Amiga. All Amiga
models imply a default CPU model, so you only need to use this option if
want to use another CPU.
