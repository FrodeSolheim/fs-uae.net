Specify the monitor to use for fullscreen.
Only implemented for the default "fullscreen desktop" mode and the
fullscreen window mode.

Value: left ("Left")
Value: middle-left ("Middle Left")
Value: middle-right ("Middle Right")
Value: right ("Right")
